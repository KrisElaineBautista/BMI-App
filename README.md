# BMI Calculator App

A simple Android BMI Calculator built with Kotlin.

## Features
- Input first name, middle initial, and surname
- Select gender
- Enter height and weight
- Calculates BMI instantly
- Shows BMI status:
  - Underweight
  - Healthy Weight
  - Overweight
  - Obese
- Gives health suggestions based on BMI

## Tech Stack
- Kotlin
- Android Studio
- XML Layouts
